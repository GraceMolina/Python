# Python code
This repository contains the following Python files:
* Functions to convert a list of tuples to a dictionary, and a dictionary to a list of tuples.
* Code to read a CSV file as a list of dictionaries and as a dictionary of dictionaries, and code to write a list of dictionaries as a CSV file.
* Code to read a CSV file as a list of lists, and code to write a list of lists as a CSV file.
* Code to calculate the age of a person in days.
* A function that returns the trace of a square matrix.
